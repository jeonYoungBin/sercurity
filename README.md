# sercurity
모든 API는 Swagger 테스트 가능 
1. JWT TOKEN 이용하여 회원가입/로그인 구현
   - 개인정보 정보 암호화
   - 특정 회원만 가입 가능("홍길동","900101-1111111")
3. JPA, H2 디비 사용
4. RestTemplete 외부 api 호출 로직 구현 -> feign 변경 예정
5. 파일 업로드 기능 구현중
